# Sanchari — Travel Landing Page

Small static website for a travel brand showcasing destinations, about and contact pages.

## Contents
- HTML pages: index.html, about.html, destinations.html, contact.html  
- Styles: css/style.css  
- Images: images/*

## Features
- Responsive landing layout
- Destinations gallery
- Contact form (Netlify-ready markup)
- Simple static setup (no build step)

## Tech
- HTML5, CSS3
- Optional: Python http.server or any static host

## Run locally (macOS)
```bash
cd /Users/shashank/Downloads/sanchari
# quick local server
python3 -m http.server 8000 --directory .
# open in browser
open http://localhost:8000
```
